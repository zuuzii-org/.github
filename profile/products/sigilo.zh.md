## Sigilo 是什么

Sigilo 是一款隐私优先的两步验证(2FA)App,生成你安全登录所需的一次性验证码 —— **TOTP 与 HOTP**,适配 Google、GitHub、微软、Dropbox、AWS 等所有支持标准验证器的服务。

它只围绕一个理念设计:**你的数据属于你。** 验证码默认只存在本机,只有你主动开启同步时,才会离开设备。

## 默认本地优先

你添加的验证码只保存在这台 iPhone 上。只要不开启同步,它就**永远不会离开你的设备** —— 无需账号,也无需信任任何云端。

想多设备使用,或换机时有个备份?开启 **可选云同步**(支持 Apple、Google、GitHub 登录)即可。它始终由你掌控,只在你发起时运行。

## 一个验证器该有的一切

| | |
|---|---|
| 🔑 **TOTP 与 HOTP** | 适配所有支持标准验证器的服务。 |
| ➕ **三种添加方式** | 扫描二维码、粘贴 `otpauth` 链接,或手动输入密钥。 |
| 🗂️ **文件夹与搜索** | 用文件夹整理,用搜索秒速定位任意验证码。 |
| 🆔 **Face ID 锁** | Face ID / 密码锁,让验证码只有你能看见。 |
| 💾 **加密备份** | 导出可自行恢复的加密备份文件。 |
| ☁️ **可选同步** | 端到端、跨设备、默认关闭 —— 完全由你决定。 |

## 值得信赖的隐私

- **永不展示广告**
- **不接入第三方追踪或分析 SDK**
- **绝不出售你的数据**
- **随时在 App 内一键注销账号** —— 永久删除账号及其在服务器上的同步数据
