## What it is · 这是什么

A local LLM API gateway. Point any client at one local endpoint, and Token Share translates between the OpenAI and Anthropic protocols in real time — so any client can talk to any provider. Streaming is supported, and it runs locally, so your API keys never leave your machine.
<br><sub>一个本地 LLM API 网关。任何客户端只要指向同一个本地地址，Token Share 就实时在 OpenAI 和 Anthropic 协议之间互译 —— 任意客户端都能接任意服务商。支持流式，纯本地运行，API key 不出本机。</sub>

## Why it exists · 为什么做它

Every client speaks a slightly different API; switching models or providers means reconfiguring everything. Token Share normalizes it into one endpoint.
<br><sub>每个客户端的 API 都略有不同，换模型/换服务商就得到处改配置。Token Share 把这些统一成一个地址。</sub>

## Features · 功能

- **One local endpoint** for every LLM client
- **Real-time protocol translation** — OpenAI ↔ Anthropic
- **Streaming** supported end to end
- **Fully local** — keys stay on your machine

## Who it's for · 适合谁

- Developers juggling multiple models and clients
- Anyone who wants provider freedom without re-wiring every tool
