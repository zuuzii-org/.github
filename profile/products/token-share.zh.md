## 这是什么

一个本地 LLM API 网关。任何客户端只要指向同一个本地地址，Token Share 就实时在 OpenAI 和 Anthropic 协议之间互译 —— 任意客户端都能接任意服务商。支持流式，纯本地运行，API key 不出本机。

## 为什么做它

每个客户端的 API 都略有不同，换模型/换服务商就得到处改配置。Token Share 把这些统一成一个地址。

## 功能

- **一个本地地址** 接所有 LLM 客户端
- **实时协议互译** —— OpenAI ↔ Anthropic
- **端到端流式** 支持
- **纯本地** —— key 留在你机器上

## 适合谁

- 在多模型、多客户端之间来回切的开发者
- 想要服务商自由、又不想重连每个工具的人
