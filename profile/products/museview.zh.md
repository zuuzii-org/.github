## 这是什么

一款专读 Markdown 与 HTML 的阅读器，本地优先 —— 文件留在你设备上，而不是某个你管不到的云。干净预览、就地编辑、导出可打印的 PDF，AI 摘要只在你点的时候才用。

## 为什么做它

多数阅读器要么把笔记锁进云端，要么把 Markdown 渲染得很糟。MuseView 让文件归你、让阅读清爽。

## 功能

- **Markdown 与 HTML** 忠实渲染
- **本地优先** —— 文件留在设备上
- **实时预览** + 就地编辑
- **导出** 干净、可打印的 PDF
- **AI 摘要按需** —— 只在你点的时候

## 适合谁

- 重度阅读者、笔记党、研究者
- 手里有一堆 `.md` 文件、并且真的会打开的人
